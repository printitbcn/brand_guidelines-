# Print It BCN — Manual de Marca Online

Manual de marca interactivo basado en el documento *Print it BCN - Branding.pdf* (2K26).

## Ver en local

Abre una terminal en esta carpeta y ejecuta:

```bash
python3 -m http.server 8765
```

Luego visita: [http://localhost:8765](http://localhost:8765)

También puedes abrir `index.html` directamente en el navegador (doble clic).

## Estructura

```
├── index.html          # Manual completo
├── css/style.css       # Estilos (estética minimalista)
├── js/app.js           # Navegación, scroll, copiar colores
├── assets/images/      # Logos y referencias extraídas del PDF
└── Print it BCN - Branding.pdf
```

## Contenido incluido

1. **Concepto** — Manifiesto, slogan, identidad verbal  
2. **Logotipo** — Principal, contrastes, espacio de seguridad, iconotipos, uso indebido  
3. **Color** — Paleta completa con valores HEX/RGB/Pantone y matices  
4. **Tipografía** — Montserrat y Verdana con jerarquía  
5. **Visual** — Dirección visual con galería  
6. **Aplicación** — Digital, papelería, merchandising  
7. **Contacto** — Datos Print It BCN
